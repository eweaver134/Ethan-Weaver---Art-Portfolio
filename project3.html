Project 3 - Environmental Model

The goal of this project was to create an environmental model that could be implemented into the a scene for a game/animation. I chose to make an old wine barrel that is rotted. This project taught me how to move a model between Maya and sculpting softwares. It also forced me to learn about telling a story through a model, via the rotting on the barrel.
